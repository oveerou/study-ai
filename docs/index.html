ABOUT

前几月期末复习，想要一个能快速整理资料、精准回答问题、自动生成知识图谱的工具，找了几个开源的感觉总是太蠢缺点智能感，于是自己做了一个 RAG 知识库 Agent，后面又加上了网页抓取、GitHub 仓库索引、本地代码目录和多源数据渠道。

核心功能

从资料导入到知识图谱，覆盖完整的学习与问答工作流。

📥

多源资料导入

支持 PDF、DOCX、Markdown、TXT 文件上传，网页 URL 抓取，GitHub 仓库克隆索引，本地代码目录扫描，以及多种来源混合导入。

RAG
🤖

Agent 意图规划

LLM 驱动的查询理解层，自动识别 5 种意图（闲聊 / 资料清单 / 概览总结 / 全文输出 / 知识问答），智能选择检索范围和来源过滤策略。

Agent

向量检索 + 重排序

基于 Qdrant 的向量相似度检索，配合 FlashRank 交叉编码器重排序，以及 LLMChainFilter 语义过滤，确保召回质量。

RAG
📊

知识图谱生成

从选定资料中自动抽取实体关系三元组，生成交互式知识图谱可视化，支持全屏展示和关系明细表格查看。

Graph
📎

来源追踪与引用

每个回答都附带精确的来源引用，可展开查看引用片段的文件名、页码和原文内容，杜绝幻觉。

RAG
💬

对话历史管理

支持多轮对话保存与加载，新建对话时自动归档旧记录并清空索引，保持工作区整洁。

界面演示

从资料导入到知识图谱，完整的使用流程。

主界面

左侧资料导入面板，右侧知识库问答区域。支持文件、URL/GitHub、代码目录、混合导入四种方式。

多源导入

支持网页 URL 和 GitHub 仓库直接索引导入，自动抓取和解析内容。

资料就绪

导入完成后显示已索引来源，侧边栏知识图谱区域可选择来源生成图谱。

资料概览问答

Agent 自动识别概览意图，结构化总结资料主题、重点内容和知识体系。

带引用的精准回答

每个回答都标注来源文件和页码，可追溯至原文片段，杜绝大模型幻觉。

GitHub 项目导入问答

直接输入 GitHub 仓库地址，自动克隆、解析代码文件并建立索引，支持对代码库进行自然语言问答。

知识图谱生成

从选定资料中抽取实体关系三元组，生成交互式力导向图，附带关系明细表格。

知识图谱全屏展示

全屏模式下的知识图谱可视化，支持拖拽、缩放、节点高亮等交互操作。

技术栈

基于主流 AI 工程框架，本地优先，数据不出机器。

️
Streamlit
交互式 Web UI 框架，快速构建数据应用
LangChain
LLM 应用编排框架，管理 Chain / Retriever / Agent
Qdrant
高性能向量数据库，支持 HNSW 索引和 Payload 过滤
⚡
FastEmbed
轻量级 Embedding 推理引擎，ONNX Runtime 加速
🔀
FlashRank
交叉编码器重排序，提升检索结果相关性
🤖
Groq / DeepSeek
LLM 推理后端，支持 OpenAI 兼容 API
📄
PyPDFium2
PDF 解析引擎，支持多页提取和元数据读取
Poetry + pytest
依赖管理与测试框架，保证代码质量

快速开始

5 分钟本地部署，无需 Docker。

克隆项目

git clone https://github.com/your-username/study-ai.git
cd study-ai

创建虚拟环境并安装依赖

python -m venv .venv
.venv\Scripts\python.exe -m pip install -r requirements.txt

配置环境变量

复制 .env.example 为 .env，填入你的 LLM API Key（支持 Groq、DeepSeek 等 OpenAI 兼容接口）。

copy .env.example .env
# 编辑 .env 填入 OPENAI_API_KEY 和 OPENAI_BASE_URL

启动应用

.venv\Scripts\python.exe -m streamlit run app.py --server.port 8501

浏览器打开 http://localhost:8501 即可使用。

运行测试

.venv\Scripts\python.exe -m pytest -q

项目结构

模块化设计，职责清晰。

study-ai/
├── app.py # Streamlit 主界面与交互流程
├── ragbase/
│ ├── agent_planner.py # Agent 意图规划器（5种意图 + 来源选择）
│ ├── agent_responses.py # 按意图分发回答生成策略
│ ├── chain.py # LangChain RAG Chain 组装
│ ├── config.py # 全局配置（路径、模型、数据库参数）
│ ├── conversation_store.py # 本地对话历史持久化
│ ├── ingestor.py # 多源导入器（PDF/URL/GitHub/Code/Mixed）
│ ├── knowledge_graph.py # 知识图谱抽取与交互式可视化
│ ├── model.py # LLM / Embedding / Reranker 工厂
│ ├── retriever.py # 向量检索 + 来源过滤 + 重排序
│ ├── runtime.py # 向量存储生命周期管理
│ ├── session_history.py # 会话历史记录管理
│ ├── source_tools.py # 资料清单 / 全文 / 概览工具
│ └── uploader.py # 文件上传处理
├── docs/images/ # 界面截图
├── tests/ # 回归测试
├── .env.example # 环境变量模板
├── pyproject.toml # Poetry 项目配置
└── README.md # 项目说明
